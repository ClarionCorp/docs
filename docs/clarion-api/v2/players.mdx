# Player Fetching

:::warning[Protected Endpoint]

This endpoint requires authentication to access. You can learn more about that [here](/FAQ/authentication).

:::


<details>
  <summary>Quick Usage</summary>
  
  <Tabs>
    <TabItem value="url" label="URL">
      <pre><code>
      {`GET ${constants.siteUrl}/v2/players/<USERNAME/ID>`}
      </code></pre>
    </TabItem>

    <TabItem value="params" label="Parameters">
      - `cached`: Boolean
        - If true, will not check Odyssey for new data. (Much faster)
      - `region`: [Region](/clarion-api/internal-types/regions.md)
    </TabItem>
  </Tabs>
  
</details>

This endpoint replies with data on a specific player. It's basically like pulling up the stats page on a Player, but in JSON form.
